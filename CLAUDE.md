# UInya - CLAUDE.md

## プロジェクト概要

**UInya**（ユーイニャ）は、**猫モチーフのゲームUI素材に特化した**ブラウザ完結型SVGエディタです。

肉球ボタン・猫耳フレーム・しっぽゲージなど、かわいい系・猫テーマのゲームUIパーツをブラウザ上でビジュアル編集し、SVG/PNG形式でエクスポートできます。

```
"ゲームUIを作るなら、まず猫から。"
```

- サーバーサイド不要、GitHub Pages のみでホスティング
- Unity / Godot / その他ゲームエンジンへの素材持ち込みを想定
- ターゲット: 猫ゲーム開発者、かわいい系インディーゲームクリエイター、Vtuber向けゲームUI制作者

---

## ブランドイメージ

- **トーン**: かわいい × ちょっとゲーム開発者っぽい
- **カラー**: パステル系（ピンク・ラベンダー・クリーム）+ ダークUI背景
- **キャラクター**: 猫のマスコットキャラ（ロゴに使用）
- **ロゴ**: 肉球 or 猫シルエット + "UInya" のテキスト

---

## 技術スタック

| 役割 | 技術 |
|------|------|
| フレームワーク | SvelteKit（`@sveltejs/adapter-static`で静的出力） |
| 言語 | TypeScript |
| SVG編集 | `Fabric.js` |
| エクスポート | SVG直接出力 / Canvas → PNG変換 |
| スタイリング | Tailwind CSS |
| ビルド | Vite |
| ホスティング | GitHub Pages |
| CI/CD | GitHub Actions |

## 実装ルール・方針

### コード品質
- TypeScript strict モードを使用
- コンポーネントは単一責務（1コンポーネント1役割）
- 状態管理は Svelte Store に集約し、コンポーネントにロジックを書かない

### SVGエクスポートの仕様
- エクスポートサイズはデフォルト 512x512px（変更可能）
- SVGは `viewBox` 付きでスケーラブルに出力
- PNGは Canvas に描画して `toDataURL()` で生成
- 透過PNG（アルファチャンネルあり）でゲームエンジンに持ち込みやすくする

### パフォーマンス
- テンプレートは `static/templates/` に SVG ファイルとして置き、ビルド時に同梱
- 外部CDN・外部APIに依存しない（オフラインでも動く）

### デプロイ
- `main` ブランチへの push で GitHub Actions が自動ビルド → GitHub Pages へデプロイ
- `svelte.config.js` で `adapter-static` + `paths.base` を設定すること

- **全体**: ダークテーマベース（ゲーム開発者が好む）+ 猫モチーフのパステルアクセント
- **レイアウト**:
  - 左パネル: テンプレート選択（猫パーツのサムネイル一覧）
  - 中央: エディタキャンバス（メイン）
  - 右パネル: プロパティ編集（色・サイズ・角丸・シャドウ）
  - ヘッダー: ロゴ（肉球アイコン + UInya）+ エクスポートボタン
- **フォント**: 丸みのあるフォント（Nunito / M PLUS Rounded 等）

## 最初にやること（Claude Codeへの指示）

以下の順番で実装を進めてください。**Phase 1のみに集中し、動くMVPを最短で作ることを優先**してください。

1. `npm create svelte@latest` で SvelteKit プロジェクトを初期化（TypeScript・Tailwind選択）
2. `@sveltejs/adapter-static` を設定し、GitHub Pages 向けビルド構成を確認
3. `fabric` をインストールし、キャンバスに角丸矩形を描画する最小実装を作る
4. 右サイドパネルで色・サイズ・角丸を変更できるUIを実装する
5. SVG / 透過PNG ダウンロード機能を実装する
6. GitHub Actions の `deploy.yml` を作成し、自動デプロイを確認する

Phase 1が完了したら `CLAUDE.md` の該当チェックボックスを更新してから次のPhaseに進んでください。
